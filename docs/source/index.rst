ispawn documentation
====================

ispawn is a tool to spawn docker containers holding multiple services with a single command.

Features
--------

* Build Docker images with multiple services (JupyterLab, RStudio, VSCode, etc.)
* Run containers with these services
* Manage containers and images
* Configure networking and volumes
